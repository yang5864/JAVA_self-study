# java-core-study

Java 기초 학습 및 알고리즘 문제 풀이 저장소입니다.

## 구조

```
java-core-study
├── java-start       # Java 입문 (변수, 연산자, 조건문, 반복문, 배열, 메서드 등)
├── java-basic       # Java 기본 (클래스, 객체지향 등)
├── BOJ              # 백준 온라인 저지 풀이
└── SWEA             # Samsung SW Expert Academy 풀이
    ├── D2
    ├── D3
    ├── baekjoon
    └── ssafy_기출
```

## 학습 내용

### Java 기초 (`java-start`, `java-basic`)
김영한 Java 강의 기반으로 학습

| 주제 | 내용 |
|------|------|
| 변수 | 기본 타입, 형변환 |
| 연산자 | 산술, 비교, 논리 |
| 조건문 | if, switch |
| 반복문 | for, while |
| 배열 | 1차원, 2차원 배열 |
| 메서드 | 선언, 호출, 스코프 |
| 클래스 | 객체 생성, 필드, 생성자 |
